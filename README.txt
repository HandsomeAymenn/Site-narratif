Lien du projet : https://handsomeaymenn.github.io/Site-narratif/

ce site a etais réaliser dans le cadre d'un projet scolaire en groupe, le but etant de réaliser un site narratif en 4 jours, (l'histoire que j'ai choisie de raconter etais Gulliver) 
La page d'acceuil a etais entierement réaliser par moi, les pages suivante ont etais réaliser par les membres de mon groupe ( je m'excuse donc du manque de contenue/serieux dans celle ci)
Je vous invite a prendre votre temps car l'apparation de l'histoire est ralentie via des propriété CSS afin de laissez le temps au lecteur de lire.
